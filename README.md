# Quick Symbols

一个用于生成苹果输入法自定义短语文件的静态网页工具，可以让你快速输入大量输入法原本没有自带的常用符号。

在线体验：[Quick Symbols](https://qsymbols.alicedrop.site)

包含常用特殊符号、单位、数学符号、集合逻辑、箭头、希腊字母等，按分类整理成可选项，用户勾选后即可一键导出 plist，并导入到 macOS 输入法中使用。

## 功能

- 按分类浏览和搜索特殊符号
- 自定义快捷码前缀，默认推荐使用 `u`
- 支持一个符号绑定多个触发词，例如同一个符号可同时支持拼音和英文
- 实时预览生成的 plist 内容
- 浏览器直接下载导出的 plist 文件

## 使用方式

1. 在项目目录启动本地静态服务：

   ```bash
   python3 -m http.server 8000
   ```

2. 在浏览器打开：

   ```text
   http://localhost:8000/
   ```

3. 在页面中：
   - 选择需要的符号分类
   - 勾选想导出的符号
   - 设置快捷码前缀和文件名
   - 点击“生成并下载 plist”

4. 将下载得到的 plist 导入或合并到 macOS 输入法自定义短语中使用

## 快捷码规则

- 为避免与拼音或英文输入冲突，建议添加前缀，默认使用 `u` 为前缀。
- 例如：
  - `σ` 可设置为 `usigma`
  - `∧` 可同时支持 `uyu` 和 `uand`
  - `∨` 可同时支持 `uhuo` 和 `uor`

## 项目结构

- `index.html`：页面结构
- `style.css`：页面样式
- `script.js`：符号数据、交互逻辑、plist 生成与下载逻辑
